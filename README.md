# DATOS-PARA-LA-TESIS-DE-MAESTRIA
Datos de viento zonal y de temperatura del planeta Marte desde 40km y hasta 80km
Las tablas de datos aquí expuestas tienen datos generados el modelo de la ESA MCD, en este caso se analizan las alturas desde 40km y hasta 80km, en 288 períodos de mediso meses marcianos (algunos de 45 días y otros meses de 60 días). La primera columan hace referencia a 288 medios meses (no son quincenas), la segunda hace referencia  las alturas entre 40000 m y 80000 m, el resto son las columnas de datos de viento zonal ó de temperatura en promeido para cada meido mes analizado, en total son 288 columnas (una para cada medio mes). Se analizan en este caso datos desde el año marciano 24 y hasta el año marciano 35 (un total de 12 años, que son 144 meses y que a su vez son 288 medios meses). Se analizan longitudes planetarias cada 60° desde el meridiano 0.
